# Git & GitHub
### What I Learned During My Web Development Internship

## Introduction

During my web development internship, one of the most important tools I learned was **Git and GitHub**. These tools help developers track code changes, collaborate with teams, and manage projects efficiently.

- **Git** → A Version Control System used to track changes in code.
- **GitHub** → A cloud platform used to store and manage Git repositories.

---

## Why Git is Important

Before Git, developers had many problems such as:

- Overwriting each other's code
- Losing previous versions of projects
- Difficulty collaborating with teams

Git solves these problems by:

- Tracking every change made to code
- Maintaining complete project history
- Allowing multiple developers to work on the same project

---

## Basic Git Workflow

The basic workflow when using Git is:

1. Create or initialize a repository
2. Add files to staging area
3. Commit the changes
4. Push the code to GitHub

### Example Commands

```bash
git init
git add .
git commit -m "Initial commit"
